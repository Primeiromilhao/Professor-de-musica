# 🎻 HERMES Violin Lab v4.6

**Plano de Prática Inteligente para Violino** — Suzuki + Ševčík + Kayser + Repertório Real

## ✨ Funcionalidades

- **Pauta Musical em Tela Branca** — VexFlow com graus funcionais (Tónica, Dominante, Sensível…)
- **Escala Ascendente e Descendente** — geração matemática perfeita sem saltos de oitava
- **Áudio PolySynth** — Tone.js offline, sem dependências externas
- **Métodos de Estudo** — Ševčík Op.1, Kayser Op.20, Flesch, Galamian, Suzuki Vol.1
- **Método Suzuki** — 6 passos guiados (Ouvir → Cantar → Imitar → Repetir → Tocar → Revisar)
- **Acordes de Sustentação** — tabela de graus com playback interativo
- **Círculo das Quintas** — navegação interativa entre tonalidades
- **Exportação WAV + PDF** da pauta
- **Dashboard de Progresso** — XP, Streak, Rank, Tonalidades dominadas

## 🚀 Como Usar

Basta abrir o ficheiro `index.html` no browser — funciona 100% offline.
